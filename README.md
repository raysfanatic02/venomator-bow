# Venomator Bow

A visual RuneLite plugin for Slayer that displays configurable text above NPCs to indicate:
- Tagged
- Poisoned
- Venomed

Defaults:
- Tagged: `T` (light blue)
- Poisoned: `P` (green)
- Venomed: `V` (bright green)

Only tracks NPCs on your current Slayer task (via VarPlayer), unless disabled in config.
